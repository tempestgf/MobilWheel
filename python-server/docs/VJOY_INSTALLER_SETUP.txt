================================================================================
                            TROUBLESHOOTING
================================================================================

PROBLEMA: "Python no está instalado"
  SOLUCIÓN: Descarga desde https://www.python.org/
            Marca: "Add Python to PATH" durante instalación

PROBLEMA: "No se puede descargar vJoy"
  SOLUCIÓN: Verifica conexión a Internet
            Descarga manualmente: https://www.vjoy.org/dl/vJoy-2.2.1.1.zip
            Extrae en: python-server/vJoy/

PROBLEMA: "Acceso denegado"
  SOLUCIÓN: Ejecuta como Administrador (click derecho en .bat)
            O: Abre CMD como Admin y ejecuta

PROBLEMA: "El instalador falla"
  SOLUCIÓN: Ejecuta: python test_vjoy_system.py
            Verifica que todos los archivos estén presentes
            Lee los logs en VJOY_INSTALLER_README.md
